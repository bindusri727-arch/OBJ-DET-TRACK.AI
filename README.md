# AI Object Detection and Tracking

## Description
This project detects objects in images using TensorFlow.js and the COCO-SSD model. Detected objects are highlighted with bounding boxes and labels.

## Features
- Image Upload
- Object Detection
- Bounding Boxes
- Object Labels

## Technologies Used
- HTML
- CSS
- JavaScript
- TensorFlow.js
- COCO-SSD Model

## How to Run
1. Download the project.
2. Open index.html.
3. Upload an image.
4. Click Detect Objects.
